# Biblioteca em TypeScript

Um projeto de biblioteca simples desenvolvido em **TypeScript**, com funcionalidades para:  
- Listar livros disponíveis  
- Buscar livros por título ou ISBN  
- Registrar empréstimos  
- Calcular multas por atraso  

Este projeto é um bom exemplo de uso de **TypeScript**, **Node.js** e organização de pastas (`src` e `dist`).
